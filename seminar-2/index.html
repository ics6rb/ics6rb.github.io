Переменные

  • var
  • let
  • const
  • Чем let отличается от const
НИКОГДА не используйте var. Вообще. Серьезно.
Всегда используйте const, если не получается (пример - цикл for) - let.

Разберем сложный пример

            
function foo(max) {
    for (var i = 0; i < max; i++) {
        setTimeout(function() {console.log(i + 1)}, i * 1000);
    }
}

foo(3);

Починим

              
function foo(max) {
    for (let i = 0; i < max; i++) {
        setTimeout(function() {console.log(i + 1)}, i * 1000);
    }
}

foo(3);

Всплытие функций
