Ruby

  • программа - текст на естетственном языке;
  • есть больше одного способа решить задачу;
  • выразительность важнее скорости.

Программа - текст на естественном языке

  • не нужно писать return;
  • не нужно писать скобки при вызове методов.

Соглашения (1)

  • my_var - просто переменная;
  • @my_var - атрибут объекта;
  • @@my_var - атрибут класса;
  • My_var или MY_VAR - константа;
  • $global - глобальная переменная.

Соглашения (2)

  • obj.method! - изменит объект;
  • obj.method? - вернет Boolean.

Тесты - как?

Обдумать

  • Посмотреть на аргументы функции и на возвращаемое значение.
  • Выделить классы эквивалентности.
  • Выделить граничные условия.

Пример

              
def divide a, b
  a / b
end

Классы эквивалентности

  • a = 4, b = 2.
  • a = 1, b = 2.
  • a = 1, b = 2.0.
  • a = 0, b = 2.
  • a = -4, b = 2.
  • a = -4, b = -2.

Граничные условия

  • a = ?, b = 0.

Написать

  • minitest (по умолчанию);
  • rspec (надо отдельно ставить).

assert

  • написанная за вас проверка (пример: если не равно, кинуть исключение);
  • ASSERT ДОЛЖЕН СООТВЕТСТВОВАТЬ ПРОВЕРЯЕМОМУ УСЛОВИЮ.

Примеры

  • Плохо (сообщение об ошибке "true != false") assert_equal true, @obj.some_method?;
  • Хорошо assert_true @obj.some_method?;
  • Хорошо assert_equal 'value' @obj.other_method.
