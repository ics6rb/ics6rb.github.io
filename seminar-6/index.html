Основы Ruby on Rails

Как все работает в вебе - если вы забыли

  • Клиент - запрашивает информацию. Всегда является инициатором
  • Сервер - отвечает на запрос

Как все работает в вебе - если вы забыли (1)

  • Клиент - это браузер
  • Клиент - это мобильное приложение
  • Клиент - это консольная программа
  • Клиент - это веб-сервис

Как все работает в вебе - если вы забыли (2) Новый файл.txt

  • Сервер - это компьютер, на который все ходят за контентом
  • Сервер - это программа, которая установлена на таком компьютере и умеет отдавать контент

Контент

  • Контент - это данные
  • Контент можно делить по "сущностям предметной области" - тогда это будут ресурсы

Простой пример - скачать фото. Контент статический

Сложный пример - зайти на страницу пользователя. Контент динамический

class HelloWorld

Ruby on Rails - MVC фреймворк над Rack и CGI

Фреймворк

  • Написанная за вас абстракция
  • Каркас для разработки
  • Имеет встроенные средства для работы с HTML и базами данных

MVC

  • Модель (Model) - для данных и логики*
  • Представление (View) - в нашем случае шаблонизатор HTML
  • Контроллер (Controller) - связующее звено, которое берет информацию из нужных моделей и передает в представление
